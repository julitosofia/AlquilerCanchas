Sistema de gestion de alquiler de canchas y compras deportivas.

Caracteristicas principales:
-Login con roles: diferencia entre clientes y empleados.
-Reserva de canchas: Seleccion por tipo, fecha y horario con validacion de disponibilidad.
-Carrito de compras: productos deportivos con control de stock y ticket final.
-Panel de estadisticas: gestion de productos,canchas y reservas.
-Arquitectura por capas: separacion en DAL, BLL y BE.


Tecnologias utilizadas:
-UI
-C#
-SQL SERVER


Como ejecutar el proyecto:
1)Clonar repositorio git clone https://github.com/tu-usuario/tu-repositorio.git
2)Abrir la solucion en visual studio.
3)Configurar la cadena de conexion en Conexion.Cadena.
4)Ejecutar el script SQL para crear las tablas y datos iniciales.
5)Compilar y ejecutar el proyecto.
